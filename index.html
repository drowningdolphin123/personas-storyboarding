Navigating the New Washing Machines in Hegeman Hall

Observations Experiences Persona A Persona B

On September 22, 2022, Hegeman Hall recieved four new washers and four new dryers. The new machines were meant to be better in every way to their predecesors with better functionality and more signage to direct users how to operate them. After observing Hegeman residents, it seems residents have had varying experiences with the new machines, depicted in Persona A and Persona B.

This description of the machine will begin at the top of the drawing and move left to right and down.

The top left element is a screen that displays messages including "Close Door", "Start", "Open Door", and "Error" as well as the time left in the cycle. The following table describes what each display means.

Close Door Displayed when the user has set wash settings and the washing machine is ready to start but the door is not closed yet.
Start Displayed when the wash settings are set and the door is closed.
Open Door Displayed when the machine has completed its wash cycle.
Error Displayed when the machine is malfunctioning or out of order.

The element on the top right is a control panel for the actual settings and preferences the user can set for their clothes. The following table displays the layout and buttons on the machines.

Wash Regular Deluxe Ultra
Cycle Normal Delicate
Temp Cold Warm Hot Start

The "Do Not Fill Above Fill Line" sticker is a reminder to users on the top of the machine. There is a faint etched line on the door of the machine (not pictured here) that shows where the machine can be filled to. Unfortunately, the line is so faint that users frequently do not see it. Thus, the reminder often does not do much to dissuade users from over-filling the machine.

The close door display is similar to the upper left element where it tells you to open or close the door depending on where the machine is in the cycle.

The "Leave Open When Done" sticker is a reminder to users on the front of the machine. The goal is to prevent mold from growing inside the machine by reminding users to keep the washer open once the cycle is finished to allow it to dry out.

The soap tray has a prominent sticker on it titled "No Pods In Tray" which can be confusing to some users. The tray takes in liquid and powder detergents, but cannot take in TidePods or similar "pod" products. Some users interpreted this sticker as "no tide pods allowed in machine" which caused some panic.
